# Football Tracker - Mobile App

This is a placeholder for the React Native / Expo mobile application.

## Getting Started

```bash
# Install dependencies
pnpm install

# Start the development server
pnpm dev:mobile

# Or directly
cd apps/mobile
pnpm start
```

## Features (Planned)

- View live match scores
- Manage watchlist
- Receive push notifications for goals
- Sync preferences across devices

## Shared Types

This app consumes types from `@football-tracker/shared` package, ensuring type safety across the entire monorepo.
